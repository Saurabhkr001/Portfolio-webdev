# Saurabh Kumar — Software Developer Portfolio 🌐

Welcome to my personal portfolio website repository!  
This project showcases my skills, featured projects, and professional experience as a Software Developer specialising in Ruby on Rails, Python, and Java.

---

## 🚀 Features
- **Modern UI/UX:** Dark theme with animated mesh backgrounds, noise grain overlay, and glassmorphism elements.
- **Interactive Elements:** Typing effect on the hero section, scroll reveal animations, and floating badges.
- **Projects Showcase:** Dynamic filtering system to sort projects by tech stack (All, Rails, Python, Java).
- **Experience Timeline:** Visual timeline of professional and academic background.
- **Contact Form:** Integrated contact form ready to be connected to services like Formspree.
- **Responsive Design:** Fully mobile-friendly layout with a custom hamburger navigation menu.

---

## 🛠️ Built With
- **HTML5:** Semantic structure and accessibility.
- **CSS3:** Custom properties (variables), Flexbox, CSS Grid, and keyframe animations (separated into `style.css`).
- **JavaScript (ES6+):** Vanilla JS for DOM manipulation, IntersectionObserver for scroll animations, and event listeners (separated into `script.js`).
- **Font Awesome 6.5.0:** For scalable vector icons.
- **Google Fonts:** 'Syne' for headings and 'DM Sans' for body text.

## 💻 How to Run Locally

1. Clone the repository: `git clone https://github.com/Saurabhkr001/<your-repo-name>.git`
2. Navigate to the project directory.
3. Open `index.html` in your favorite web browser. No build steps or local server required!
